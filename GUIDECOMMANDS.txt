Доступные команды:

pwd        - показать текущую директорию
ls         - показать содержимое
cd <directory>   - перейти в директорию
mkdir <directory>  - создать директорию
cat <file>    - вывести файл
head <file> n - первые n строк файла
tac <file>    - вывести файл наоборот
tree       - показать структуру директорий
rm <folder>     - удалить файл или пустую папку
